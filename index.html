My Work

Flint Industries Full Website (2012)

Flint Industries is an international industry leader in geotextile manufacturing. I created this site from scratch using HTML5, CSS, JavaScript, jQuery, Google Web Fonts, and PHP.

To create the site graphics and initial mockups, I used Adobe Photoshop and Illustrator. On the home page, I used the Smooth Div Scroll jQuery plugin to show scrolling images. The drop-down navigation was made using the MooTools JavaScript framework MenuMatic plugin. Videos throughout the site were embedded using Flowplayer.

The Contact page features an interactive JavaScript map which I coded. Finally, forms were made using the FormM@iler PHP script.

The site put Flint's online presence ahead of its competitors, halved call times, and has resulted in a noticeable increase on international interest.

G&R Farms Updated & Expanded (2012)

G&R Farms is a Vidalia onion farm located in Glennville, GA. I provided numerous updates to content on the site, using HTML, CSS, and jQuery.

Phrasal Template Game (2016)

This is a "Mad Libs" style game that I created from scratch using HTML5, CSS3, Google Web Fonts, JavaScript, and Regex.

The game's forms are generated dynamically from story strings containing place holder values such as [Adjective]. JavaScript and Regex are used to find these placeholder values and generate the form. On submission, the placeholders are replaced in the story string and the story is shown.

CSS3 is used extensively to provide a consistent theme and to animate elements of the game. The form features basic validation to ensure all fields are populated before submitting. Responsive design has been used to ensure the game is playable on desktop and mobile devices.
